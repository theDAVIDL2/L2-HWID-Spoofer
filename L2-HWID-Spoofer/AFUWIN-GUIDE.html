Step 1: Dump bios.rom

First, unzip the AFUWIN tool, then open an Administrator Command Prompt.

1

Navigate to AFUWIN folder

cd "C:\path\to\AFUWIN"

Replace with your actual AFUWIN path

2

Dump the BIOS

AFUWINx64.exe bios.rom /O

The /O flag means "Output" - this creates a dump of your current BIOS

Note: During the BIOS reading process, you may not be able to move the mouse, and sound may become buggy. This is normal.

If everything went correctly, you should now have a bios.rom file in the AFUWIN directory.

Step 3: Editing Your Serials

This is the most important step. You will modify multiple serials using DMIEdit and HxD.

üîß System Serial Number (DMIEdit)
  1. In DMIEdit, go to [Type 001] ‚Äî System Information
  2. Double-click "Serial Number"
  3. Under "String Value: xxx", change your Serial Number to something new
  4. Click the lightning bolt icon labeled "All" to apply changes
üîß CPU Serial Number (DMIEdit)
  1. In DMIEdit, go to [Type 004] ‚Äî Processor Information
  2. Double-click "Serial Number"
  3. Under "String Value: xxx", change your Serial Number
  4. Click the lightning bolt icon labeled "All" to apply
üîß System UUID (HxD Required)

The UUID is more complex and requires hex editing:

  1. In DMIEdit, go to [Type 001] ‚Äî System Information
  2. Double-click "UUID"
  3. Copy your current UUID from "Hexadecimal Value: xxx"
  4. In HxD with bios.rom loaded, press Ctrl + R to open Replace
  5. Select "Hex-values" mode
  6. In "Search for:", paste your current UUID
  7. Go back to DMIEdit, press "Auto Generate system UUID"
  8. Copy the new UUID
  9. In HxD "Replace with:", paste the new UUID
  10. Under "Search direction" select "All"
  11. Click "Replace All"
  12. Press Ctrl + S to save
üîß Motherboard Serial Number (HxD Required)
Important: Your new serial number MUST be the same length as the original!
Example: Old Serial: 12345678 ‚Üí New Serial: 87654321 (both 8 characters)
  1. In DMIEdit, go to [Type 002] ‚Äî Base Board/Module Information
  2. Double-click "Serial Number" and copy your current serial
  3. In HxD with bios.rom loaded, press Ctrl + R
  4. Select "Text-string" mode
  5. In "Search for:", paste your current Serial Number
  6. In "Replace with:", type a new serial of the same length
  7. Under "Search direction" select "All"
  8. Click "Replace All"
  9. Press Ctrl + S to save

1

2

If you followed all steps correctly, you should see a success message. Your modified BIOS has been written to the motherboard.
