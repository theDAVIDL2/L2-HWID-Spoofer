========================================
  TROUBLESHOOTING
========================================

If USB won't boot:
 → Check Secure Boot is ENABLED (not disabled)
 → Try USB 2.0 port
 → Disable Fast Boot in BIOS

If system won't boot after install:
 → Run 07-Emergency/emergency-restore.ps1
 → Or select "Windows Boot Manager" in BIOS

If HWIDs not changing:
 → Verify spoofer configuration
 → Check spoofer ran (boot time +2 seconds)
